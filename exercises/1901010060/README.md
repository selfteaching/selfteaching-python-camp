举个栗子：在计算器这个编程中，最主要就用到if elif else语句
if expression1:
        statement1
    elif expression2:
        statement2
    elif expression3:
        statement3
    else expression4:
        statement4
那我们就是首先，不断训练if语句的运用，加深了if语句的理解后，配合语句、函数就能更好的运用了。
