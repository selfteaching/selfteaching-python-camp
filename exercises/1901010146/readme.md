终于来了有意思的任务，在编程中遇到了以下几个问题：

### 任务一：使用for...in循环打印九九乘法表
1. **print不换行问题：** 通过这次作业，我发现官方文档介绍的一点都不详细，print()的参数根本没有介绍啊，害得我又找视频、又找教程，才发现了print(end=' ')的用法；
2. **条件语句后面的“：”。**这是个语法问题，现在知道了当下面要写包含的程序时，就要加冒号
3. **for语句的使用，range（）**，结束值的问题
4. if语句的使用：==，break
5. **如何解决3x3与4x3错位的问题。** 这完全是因为我强迫症，pdf里的例子也是错位的，但我就是想把它对齐了。开始也走了弯路，想那里错位就调哪里，后来发现太不智能了。最后，想到把所有个位数的结果前面都价格空格，这样结果都占两位就齐了。也就是加了一个判断语句：
```
a = i*j
if a < 10:	
	print(i,'*',j,'= ',i*j,'  ',end='')
else:
    print(i,'*',j,'=',i*j,'  ',end='')
```
完整程序如下：
```
i=0
j=1
for i in range(i,10,1):		#结束值由9变成了10，说明值为9时，数值只停止到8
	j=1
	print()		#单纯的换行功能
	for j in range(j,i+1,1): 	 #i+1解决了没有n*n的问题
		a = i*j
		if a < 10:		#这个条件语句，解决了输出错位的问题
			print(i,'*',j,'= ',i*j,'  ',end='')
		else:
			print(i,'*',j,'=',i*j,'  ',end='')
```	
### 任务二：使用while循环打印九九乘法表并用条件判断把偶数去除掉
作业中，主要思考if与while的区别是什么，基本上if能实现的功能，使用while一样能够实现，那么哪种情况下，是不可代替的呐？
我觉得while可以实现无限循环比如：``` while 1```，而for必须先判断，才能执行。

完成的程序如下：
```
i = 1
j = 1
while i < 10:
	j = 1
	print()
	while j <= i:
		a = i * j
		if a < 10:		#这个条件语句，解决了输出错位的问题
			print(i,'*',j,'= ',i*j,'  ',end='')
		else:
			print(i,'*',j,'=',i*j,'  ',end='')
		j += 1
	i += 1
```
在作业过程中，还偶然想到如何使用print()一次性输出n个空行的问题，通过百度发现了```print('\n'*n)```的用法，很有意思!
