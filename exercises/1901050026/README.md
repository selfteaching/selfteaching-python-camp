学习总结
  之所以报名python自学营就是为了锻炼自己的自学能力，感觉自从走出校门之后，自学能力已逐渐退化为0，认为不靠别人手把手的教，学会一门技能是不可能的事情。

  学习开始之初以为学习python应该并不特别难，但是实际的情况远远超出我的预估，尤其是看了很多遍技术文档之后，仍然一脸茫然，甚至开始怀疑人生，直到阅读很多同学的打卡笔记和issue后，才恍然大悟其实有这样的经历何止我一个，学习本来就是一个从笨拙到熟练的内化过程，之所以笑来老师号召大家快速入门，目的是要求短期快速掌握最少必要知识，完成从一个门外汉到会简单应用，初级入门的初学者的转变。

  通过13课时的学习，收获最大的是提升了自我的自学能力，从不喜欢自己独立思考寻找解决方案，思考之后再请教他人，不是上来就问，从被动到主动式学习的转变，学习过程中也暴露出很多不足，教是最好的学没有做到，没有用自己懂得的知识主动帮助他人，学习笔记记得也不够认真,没有达到检验所学知识的目的。python课程中很多知识点还没有吃透，后期需要继续学习MIT python课程循序渐进的提升技能。最后感谢教练和辅导员给予的支持和鼓励，你们的帮助对我很重要，让我度过了最痛苦的时光，未来的学习之路还很漫长，这段经历使我受益匪浅。没有什么学不会没有什么学不成，用实际行动不断践行，朝着这一目标不断迈进。
