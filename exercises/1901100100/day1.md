# firstday
13日教练温宇华给我发了第一天的作业，还有视频。说实话，视频声音小到无法听清楚。在手机上也看不清楚。我刚刚提了问题，又想起“how to ask question the smart way"。于是自己谷歌搜索解决（下载桌面钉钉分享文件到电脑），下载暴风影音解决声音小的问题。虽然，每个人都要解决问题，但是这多少还是影响体验，还有花费了时间在编程以外的事情上，不是很好。

15日，按照视频的指引，学会如何在github上进行操作。建立master和develop分支。明天在安装GitHub桌面版吧。

16日晚上，登录github，安装github桌面版程序，安装过程很顺利，按照工作任务的要求建立文件，pull request。等等都很顺利。
但是到了clone仓库到本地的时候，麻烦大了。不知道为什么，clone的速度最高只有14kb，最慢2-4kb，但是我家里是光纤宽带，500M。17日上午，温宇华告诉我一个方法加快速度，https://blog.csdn.net/w958660278/article/details/81161224
17日晚上进行设置，结果发现win10操作系统默认不是系统管理员权限，无法更改hosts文件。刚刚安装了win10家庭版操作系统，可恶的比尔盖茨的家庭版功能还有不少欠缺。比如没有用户组的设置。经过一段时间的搜索答案，终于找到如何进入另一个administrator账号，更改了hosts文件。重新刷新vdn。
这时候，克隆仓库的速度快了很多，没有多久就下载结束。然后在本地新建文件，subsummit. pull request 完成day1 的作业。
