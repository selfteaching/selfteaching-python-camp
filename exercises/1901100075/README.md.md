前言：作为一名准大一生，我的专业就是计算机。我是属于那种overthinking的人，做事情总会再三犹豫，特别倾向于听取别人的意见。但是，很多人都说学计算机会秃头，会猝死......反正一大堆理由让我劝退，甚至怀疑自己的决心。后来又转念一想，他人之所以有这样的看法，是因为他经历过啊！况且，学习计算机本来就是我的梦想，它的好与坏，适不适合我，总得经历才能下决定。后来，我很有幸能阅读到笑来老师写的《为什么把编程当成自学的入口？》，其中提及到的“到了最近，不懂数据分析的，也开始算作文盲了。”更是让我察觉到深深的恐惧，一度在想，如果再不学编程，是不是就只能眼睁睁地看着自己被数据支配，被飞速发展社会淘汰？于是，我下定决心报名python的入门训练营。

总结：
1、收获与感受
语言学家曾经说过：“如果我们脑子里没有一个概念的话，我们的脑子就倾向于不去想那件事情.....”经历过十四天的埋头苦干和风风雨雨，我了解到python 的基本框架和概念，学会阅读官方文档来制作计算器，学会进行流程控制并注意一些标点符号和缩进，学会使用模块，第三方库......这一路走来，磕磕碰碰，但回头一看，却是令人值得留恋的风景，我觉得我已经变成更好的自己。
2、遇到的难点
一、制作day3的计算器，我毫无思绪。当我沉浸在day2学会写第一行代码的喜悦之中，陶醉于自己美好的想象世界里―我会变得很厉害！带着足够多的信心去着手完成day3的任务，结果现实给我一记当头棒喝，这两天来任凭我怎么看官方文档，我也不知道怎么做一个calculator，于是就去问助教，助教告诉我，先要思考怎样才能让计算机正常操作，然后根据需要选择函数。听完建议的我更是一头雾水。没办法，只好对照视频慢慢摸索。毕竟，刚接触python的我，对执行代码不太理解，只能敷衍得完成任务。

二、day5的三座大山分分钟想劝退我，统计词频，数据转换，进制变换......没办法，只能一个个上了。day4还完成得顺风顺水，day5就直接放大招，甚至还产生一个想法―我是不是完不成这十四天的任务了？这时，脑海突然闪过笑来老师说过的“麻烦守恒定律”，有些麻烦是不能逃避的，逃得了初一，逃不过十五。你现在偷过的懒，未来会加倍奉还......于是，硬着头皮，在助教，教练的协力帮助下，对照视频，查阅单词和官方文档，一步步地分解。在完成的时候还开心了好半天,在实践当中也深化对python的认知。
