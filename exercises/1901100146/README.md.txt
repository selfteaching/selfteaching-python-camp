 # day1自学心得

# 我学习了github操作系统，并且会简单创建自己的代码库。系统虽然不太熟悉，但是经过两次重新操作，终于有了初步的印象并且比第一次操作要快很多。

在学习过程中遇到很多单词不认识，甚至不知道自己这么操作是什么意思。先跟着教练一步一步实操，不懂的问题直接问助教，目前第一天学习还是比较顺利的。

# 自学python，让我领悟了三个道理

第一，要有自我解决问题的能力。比如不知道自己那里操作错了，就去百度找答案、根据经验一点一点的改正过来。比如刚入门的时候，我的电脑就遇到很多问题插入耳机没声音，自己百度解决了。不知道作业什么意思，就直接截图问助教。以前遇到问题，总是会发慌，现在自学后，反而冷静很多，会自己想办法去解决.

第二，提升耐心力。以前只要学习技能遇到很多困难，会直接放弃，一点耐心都无现在会沉下来一点一点跟着教练学习，比如第一遍练习，全程蒙蔽状态，第二遍就好多了而且能耐心做2个小时，搞懂每一步操作意思，不懂的地方用笔记记录下来。

 第三，提升专注力

我发现只要一打开课程开始学习，就会忘记时间存在，沉浸学习。感觉下午的时间比晚上过的快，主要是晚上看电视和浏览新闻多，感觉身体被掏空。可下午固定时间学习，就很专注了。而且我对python保持强烈好奇心，主要原因是重来没接触过。

 之前电脑坏了，进度条松下来了，今天电脑好了，赶紧补上第一天作业，希望接下来14天每天跟上战友脚步，早日入门。之后我会去网上找一些爬虫学习的视频来学习，尽快学好python，用在自己的工作中。
