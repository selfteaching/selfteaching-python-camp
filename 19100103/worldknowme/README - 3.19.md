收获总结及问题建议如下：

①VSCODE最基本的工具之一：CTRL+shift+P ，可以在里面调用python select interpreter等

②学会用print。在新建的d2_exercise_hello_python.py中，利用print("hello world")为例，可以做出简单的输出。


③python具由输入提示功能，如：输入print只需输入pr后，python就可帮你智能识别出pr开头的很多单词，可以用键盘的上下键/或按table按键可以完成输入
④在插件的那一栏里，可以安装python，这是课程任务，不必多说。同时可以搜索Chinese来搜索Microsoft官方版的中文工具，这个工具可将vscode汉化(汉化的目的是让我以最舒适的方式了解这个软件，当然，我也会同时用英文版的vscode)

⑤数PowerShell可以在windows中任意文件夹按住shift并右键鼠标看见打开，类似于加强版的CMD 。在VSCODE(python)中在软件也有powershell可以调用


⑥为了解决3.18第一次学习效率低下的问题，我从我最熟悉的视频教程入手，从b站上寻找学习python的中文及英文视频课程。果然，找到了虽为数不多但十分高质量，且十分适合新手的视频课程（至少我觉得它十分适合我，本人对于阅读这件事还没有达到足够感兴趣的程度，所以为了提起对于学习python的兴趣，我尽量先让自己对python的基本知识感兴趣）。由于3.19的学习任务相对3.18比较轻，所以这也让我学习了3.19任务以外的东西.
