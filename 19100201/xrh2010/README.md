想学习一门编程语言，是我小时候的一个念想，中间虽然也经历过几次开始，但终究没有坚持下去。13天的课程，我咬咬牙把课程作业做完，最后是抄完（抄的还有一大部分无法运行）。把一个循环走下来，从基本概念到做出成品。
学的过程中发现自己存在很不好的习惯，比如：
1.在搜索相关资料的过程中，经常忘记了目标，经常花费很多时间在看别的资料；
2.同时存在思维惰性，第一次做不出来后，参考了同学的代码，完成了任务，后面连续几次都不想自己动脑筋，直接抄写同学的代码；
3.发现自己原先的关于学习的想法不是很正确，以前觉得要把全部的基础知识学会了才能开始做成品，花了大量的时间在基础知识方面，花了大量精力却毫无成就感，经常半途而废，这次13天的自学，直接往成品推进，这很多程度上颠覆我的认知，从学会了再做到边学边做。这可能是我长期以来自己学习的一个误区，没有结果，没有目标的随便学习。
4.实时反馈很重要，会不会，对不对，计算机会直接告诉你。5.共同学习的伙伴很重要。一个自学太孤独，尤其是在还不具备自学能力的阶段。
还有很长的路要走。
